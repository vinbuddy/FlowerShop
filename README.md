## Technology used
 - C# + Asp net MVC
 - Web API
 - Javascript
 - Boostrap 5
 - Jquery
 - HTML, CSS
## Features
- Sign up / Sign in (form validation)
- Admin manage products and orders
- Search products
- Filter, pagination 
- Add, Remove, Update to cart
- Checkout
- Tracking order, cancel order
- Send Email
